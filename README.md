# CodeAlpha_HangmanGame

## Project Overview
Hangman Game is a word guessing game developed in Python. The player must guess the hidden word one letter at a time before running out of lives.

## Features
- Multiple word categories
- Difficulty levels
- Hint system
- ASCII Hangman graphics
- Score calculation
- High score tracking
- Input validation
- Play Again option

## Technologies Used
- Python 3

2. Open a terminal in the project folder.
